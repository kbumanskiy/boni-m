# Бони М — самоучитель азбуки Морзе (PWA)

## Что это и для кого
Игра-самоучитель приёма и передачи морзянки на слух. Для **папы Кости** — радиолюбитель,
73 года, позывной **Boney M**, не айтишник. Принципы: простота и надёжность важнее функций,
крупный шрифт, без скрытых жестов, полностью офлайн, тёплый уважительный тон.

Полное ТЗ — [`boni-m/TZ.md`](boni-m/TZ.md). Разбор/критика/план — [`PLAN.md`](PLAN.md).

## Статус
🟢 **Ядро собрано и протестировано.** Логика (44 автотеста) и интерфейс (18 проверок дымового
теста) — зелёные. Осталась приёмка на реальном Android — [`ACCEPTANCE.md`](ACCEPTANCE.md) — и деплой.

## Технологии (жёстко)
Чистые HTML+CSS+JS, **без библиотек/фреймворков/CDN/сборщиков**. Звук — Web Audio (без файлов).
PWA с офлайном (service worker, cache-first). Прогресс — `localStorage` (`boni_m_state`),
с миграцией, резервной копией и `storage.persist()`. ES-модули, поэтому те же файлы из `app/js/`
тестируются в Node.

## Принятые решения (отличия от буквального ТЗ — согласованы с Костей)
1. **Позывной «Boney M» привязан к русскому треку** (а не к латинице). Коды Б,О,Н,Е,Ы,М = B,O,N,E,Y,M,
   поэтому спецдрилл доступен после освоения Ы (21-я буква), без английского режима.
2. **Цифры подмешиваются в общий поток «Учиться»** после 20 букв (без отдельного экрана).
3. **Хостинг — GitHub Pages** (авто-деплой `app/` через `.github/workflows/deploy.yml`).
4. Знаки препинания §3.4 внесены **дословно по ТЗ**; 3 кода (Точка/Запятая/Восклицание)
   расходятся с ITU — помечено флагом в `js/data.js`, ждёт решения (не блокирует, см. ACCEPTANCE).

## Как запускать и тестировать
```bash
cd app && python3 -m http.server 8000   # посмотреть на компьютере → localhost:8000
npm test                                  # автотесты логики
node test/smoke.mjs                       # дымовой тест интерфейса (JSDOM)
```

## Структура
- `app/` — само приложение (хостится). Контроллер — `app.js`, чистая логика — `app/js/*`.
- `test/` — автотесты логики (`*.test.js`) + дымовой тест UI (`smoke.mjs`).
- `boni-m/` — исходное ТЗ и мастер-PNG ассетов (герой). Рабочие WebP/иконки — в `app/assets/`.

## Git и деплой
Ветки: `main` (прод, авто-деплой на Pages), `dev` (разработка). Правила Git — в `/root/CLAUDE.md`.
Pages включается один раз: Settings → Pages → Source: GitHub Actions. Репозиторий пока **не запушен**.
