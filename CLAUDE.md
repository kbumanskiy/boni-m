# MorseTrainer — самоучитель азбуки Морзе (игра для папы)

## Что это и для кого
Игра-самоучитель для **папы Кости** — учим азбуку Морзе в игровой форме.
Главный пользователь — пожилой человек, **не айтишник**. Значит:
- **Никакой установки.** Игра должна открываться в один клик (браузер).
- **Крупный шрифт, простой экран, минимум кнопок.**
- **Работает без интернета** (открыл файл — играешь).

## Текущий статус
🟡 **Старт.** Заведена папка, пишем первую версию. Кода пока нет.

## Выбранное направление (можно поменять)
- **Формат:** одностраничная игра в браузере (один HTML-файл со звуком на Web Audio API).
  Папе достаточно прислать/положить файл — открыл, играет. Не требует сервера и установки.
- **Методика:** метод Коха — учим по 1–2 знака за раз, на слух, с постепенным добавлением.
  Это проверенный способ выучить Морзе быстрее, чем по таблице.
- **Язык интерфейса:** русский.
- **Геймплей (черновик):** звучит знак точками-тире → папа выбирает букву → очки, прогресс,
  «уровни» по мере освоения новых знаков.

## Открытые вопросы (уточнить у Кости при следующем заходе)
- Русская азбука Морзе или латиница? (для радиолюбителя — обычно учат обе; для начала — русская)
- На чём папа будет играть: телефон или компьютер? (влияет на размер кнопок и раскладку)
- Цель: просто развлечение / хочет реально выучить (например, для радио)?

## Как запускать (заполнится, когда будет код)
Пока кода нет. Планируется: открыть `index.html` в браузере — и всё.
