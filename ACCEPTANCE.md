# Чек-лист приёмки на реальном Android

Это то, что **нельзя проверить без телефона** — прогоняет Костя (или сын). Логика, таблицы,
тайминг и сохранность уже покрыты автотестами (`npm test` — зелёные). Здесь — поведение на устройстве.

Открыть сайт на Android, добавить на главный экран, и пройти по пунктам (из ТЗ §14):

- [ ] Ставится на главный экран, открывается на весь экран.
- [ ] **Офлайн:** выключить интернет — всё работает (со второго открытия).
- [ ] Звук появляется с первого нажатия, без щелчков; тон и громкость регулируются.
- [ ] На медленной скорости паузы заметно длиннее (Фарнсуорт слышен).
- [ ] Справочник: выборочно сверить 10 знаков, обязательно Ъ, Э, Щ, Ч, Ш, Й.
- [ ] «Учиться»: новый знак открывается по правилу; прогресс цел после перезапуска.
- [ ] «Ключ»: точка/тире и границы знака ловятся без напряжения на скорости по умолчанию (12).
- [ ] «Ключ»: нет «залипания» тона.
- [ ] Реакции героя появляются/исчезают, без ореола на кремовом фоне.
- [ ] Серия «дней в эфире» растёт по дням, без укоров при пропуске.
- [ ] Спецдрилл «Boney M» доступен после освоения нужных знаков, засчитывает веху.
- [ ] Старт — сразу 4 знака; в выборе всегда ≥4 кнопок; верный ответ в случайной позиции.
- [ ] Переключение РУС↔ENG сохраняет раздельный прогресс.
- [ ] «Повторение» гоняет только освоенные знаки, новых не открывает.
- [ ] Буксующий знак → мягкое предложение «отложить»; возвращается позже; не больше двух сразу.
- [ ] Выход из занятия и возврат — тот же активный набор.
- [ ] На старте спрашивает имя; имя в приветствии; в кабинете правится имя/позывной.
- [ ] Очки растут; журнал занятий пополняется.
- [ ] **ГЛАВНОЕ:** закрыть, выгрузить из памяти, перезагрузить телефон — прогресс на месте.
- [ ] Резервная копия сохраняется в файл и успешно восстанавливается.
- [ ] Крупный шрифт, большие кнопки, высокий контраст; работает при системном увеличении шрифта.

## Открытый вопрос (не блокирует)
- В §3.4 ТЗ (необязательный раздел «Дополнительно») три кода — **Точка, Запятая, Восклицание** —
  расходятся с международным ITU. Сейчас в Справочнике они показаны **как в ТЗ**. Решить: оставить
  по ТЗ или поправить по ITU. На обучение (буквы/цифры) не влияет.
