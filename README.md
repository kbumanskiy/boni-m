# Бони М — самоучитель азбуки Морзе

PWA-игра для обучения приёму и передаче азбуки Морзе на слух. Сделана для папы
(радиолюбитель, позывной Boney M). Чистые HTML+CSS+JS, без библиотек, полностью офлайн.

- **ТЗ:** [`boni-m/TZ.md`](boni-m/TZ.md)
- **План, критика, критерии:** [`PLAN.md`](PLAN.md)
- **Чек-лист приёмки на телефоне:** [`ACCEPTANCE.md`](ACCEPTANCE.md)
- **Само приложение:** папка [`app/`](app/) — это и есть то, что хостится.

## Посмотреть на компьютере
```bash
cd app
python3 -m http.server 8000
```
Открыть `http://localhost:8000`. (PWA-установка и звук на телефоне — по HTTPS, см. деплой.)

## Тесты
```bash
npm test          # автотесты логики (таблицы, тайминг, методика, геймификация, сохранность)
node test/smoke.mjs   # дымовой тест интерфейса (рендер экранов и клики)
```

## Деплой (GitHub Pages)
В репозитории настроен авто-деплой папки `app/` на GitHub Pages при пуше в `main`
(`.github/workflows/deploy.yml`). После первого пуша включить Pages:
**Settings → Pages → Source: GitHub Actions.** Дальше всё само: пуш в `main` → сайт обновился.

Папа открывает ссылку на телефоне (Android) → меню браузера → «Добавить на главный экран».

## Структура
```
app/
  index.html, style.css, app.js   — оболочка и контроллер интерфейса
  manifest.webmanifest, sw.js     — PWA: установка и офлайн
  js/                             — чистая логика (тестируется отдельно)
    data.js     — таблицы Морзе и порядок Коха (§3, §6)
    timing.js   — тайминг PARIS + Фарнсуорт, пороги «Ключа» (§5)
    state.js    — состояние, localStorage, миграция, сохранность (§2, §13)
    progress.js — методика Коха: набор, правило 90/30/10, отложенные, цифры (§6)
    gamify.js   — ранги, серия дней, вехи, очки, сессии, позывной (§8, §9)
    audio.js    — Web Audio: тон, проигрывание, «Ключ» (§4)
  assets/                         — герой (WebP) и иконки PWA
test/                             — автотесты логики + дымовой тест UI
boni-m/                           — исходное ТЗ и мастер-PNG ассетов
```
